Инструменты

Cписок инструментов, приложений, оборудования и прочего, что я ежедневно использую для проектирования и кодирования.

Разработка

  • Использую Visual Studio Code , тема Atom One Dark и шрифтом Operator Mono
  • Chrome - мой основной браузер как для разработки, так и для общего использования.
  • React - библиотека Javascript. Компонентно-ориентированная модель мышления - это первое, что действительно впечатлило меня при знакомстве с React.
  • Для эффектов 3D и шейдеров использую three.js. С его помощью можно делать действительно мощные вещи.
  • Для Javascript анимаций использую Popmotion Pure 8, отличная библиотека для spring-анимаций. Для CSS анимаций enter/exit React Transition Group
  • Для создания и тестирования компонентов пользовательского интерфейса в изоляции я использую Storybook

Дизайн

  • Figma главный инструмент для макетов
  • Вся графика движения создается в Adobe After Effects.
  • Для 3D моделей я использую Blender. С версии 2.8 он стал намного проще в использовании и во многих отношениях лучше, чем дорогие 3DS Max или Maya.
